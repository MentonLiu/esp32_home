# 项目接线说明

本文按两个固件工程分别列出主要接线，便于装配和联调。

## 1. esp32_home_client 接线表

| 功能模块     | 代码常量                     | ESP32 引脚 | 连接对象     | 信号方向 | 说明                                   |
| ------------ | ---------------------------- | ---------- | ------------ | -------- | -------------------------------------- |
| 按钮 1       | client_config::kButton1Pin   | GPIO32     | 轻触按键一端 | 输入     | INPUT_PULLUP，按下后让窗帘状态前进一档 |
| 按钮 2       | client_config::kButton2Pin   | GPIO33     | 轻触按键一端 | 输入     | INPUT_PULLUP，按下后让风扇状态前进一档 |
| 按钮 3       | client_config::kButton3Pin   | GPIO26     | 轻触按键一端 | 输入     | INPUT_PULLUP，当前预留                 |
| 按钮 4       | client_config::kButton4Pin   | GPIO13     | 轻触按键一端 | 输入     | INPUT_PULLUP，当前预留                 |
| RGB 红色通道 | client_config::kRgbRedPin    | GPIO25     | RGB 模块 R   | 输出     | 数字输出高电平点亮（按当前实现）       |
| RGB 绿色通道 | client_config::kRgbGreenPin  | GPIO14     | RGB 模块 G   | 输出     | 数字输出                               |
| RGB 蓝色通道 | client_config::kRgbBluePin   | GPIO15     | RGB 模块 B   | 输出     | 数字输出                               |
| 传感屏 OLED SDA | client_config::kSensorOledSda | GPIO21  | OLED 传感屏 SDA | 双向  | I2C 数据线（总线 1）                    |
| 传感屏 OLED SCL | client_config::kSensorOledScl | GPIO22  | OLED 传感屏 SCL | 输出时钟 | I2C 时钟线（总线 1）               |
| 状态屏 OLED SDA | client_config::kStatusOledSda | GPIO23  | OLED 状态屏 SDA | 双向  | I2C 数据线（总线 2）                    |
| 状态屏 OLED SCL | client_config::kStatusOledScl | GPIO18  | OLED 状态屏 SCL | 输出时钟 | I2C 时钟线（总线 2）               |
| 双 OLED 地址   | client_config::kSensorOledAddress / client_config::kStatusOledAddress | 0x3C | 两块 0.96 寸 OLED | - | 两屏通过独立 I2C 总线复用同地址 |

## 2. esp32_home_server 接线表

| 功能模块       | 代码常量              | ESP32 引脚 | 连接对象              | 信号方向 | 说明                                       |
| -------------- | --------------------- | ---------- | --------------------- | -------- | ------------------------------------------ |
| DHT 温湿度数据 | pins::DHT_DATA        | GPIO13     | DHT11/22 DATA         | 输入     | 传感器数据线；电源按模块规格接 3.3V 或 5V  |
| 光照模拟量     | pins::LIGHT_ANALOG    | GPIO4      | 光敏模块 AO           | 输入     | ADC 输入，代码中按反向百分比处理           |
| MQ2 烟雾模拟量 | pins::MQ2_ANALOG      | GPIO5      | MQ2 模块 AO           | 输入     | ADC 输入，用于烟雾分级                     |
| 雨滴模拟量     | pins::RAIN_ANALOG     | GPIO6      | 雨滴模块 AO           | 输入     | ADC 输入，代码中按雨量百分比与滞回判定降雨 |
| 风扇 PWM       | pins::FAN_PWM         | GPIO18     | 风扇驱动输入端        | 输出     | 通过 LEDC 输出占空比，不可直接带大电流负载 |
| 窗帘舵机 A     | pins::CURTAIN_SERVO_A | GPIO16     | 舵机 A 信号线         | 输出     | 50Hz 舵机控制信号                          |
| 窗帘舵机 B     | pins::CURTAIN_SERVO_B | GPIO17     | 舵机 B 信号线         | 输出     | 与 A 通道反向角度联动                      |
| 蜂鸣器 PWM     | pins::BUZZER          | GPIO12     | 有源/无源蜂鸣器驱动端 | 输出     | LEDC 音调输出                              |

## 3. 联调注意事项

| 项目     | 要求                                                                                        |
| -------- | ------------------------------------------------------------------------------------------- |
| 共地     | client、server、外设模块必须共地                                                            |
| 电源     | 舵机和风扇建议独立供电，避免由开发板 3.3V 直接供大电流负载                                  |
| 按键接法 | 使用 INPUT_PULLUP 的按键，统一一端接 GPIO、一端接 GND                                       |
| I2C      | 客户端当前为双 I2C 总线双 OLED（两屏都可用 0x3C），无需在同一总线上规避地址冲突             |
| 网络约定 | client 与 server 当前都优先连接 `iPadmini / lbl450981`，失败后回退 `esp32-server / lbl450981` |
| 板型差异 | server 为 ESP32-S3，client 为 ESP32，跨板复用引脚时先核对可用 GPIO 与外设冲突               |
