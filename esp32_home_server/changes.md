# 服务端变更记录

日期：2026-03-21

## 本轮调整

1. 根据最新项目要求，将服务端 Wi-Fi 主网络固定为 `iPadmini / lbl450981`。
2. 保留 `esp32-server / lbl450981` 作为兜底热点。
3. 将红外控制协议收敛为“通过串口向 ESP8266 发送字符串命令”。
4. 本地控制接口统一到 `GET /api/status` 与 `POST /api/control`。
5. 状态接口新增 `fanPowerOn`，明确风扇电源与风速是两个状态。
6. 风扇只有在通电状态下才允许调速，自动化逻辑也遵守这一规则。
7. 删除服务端网页与项目结构页的关联，结构说明改为仓库根目录的 `project-map.html`。
8. MQTT 代码暂时保留占位能力，但当前主流程不启用。
9. 为客户端联调同步清理了时间展示依赖字段，避免双端协议出现无意义负担。
